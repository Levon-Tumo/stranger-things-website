# Stranger Things Fan Website

A dynamic and interactive website dedicated to the Netflix series Stranger Things. This website features character profiles, memorable moments, and atmospheric effects inspired by the show.

## Features

- Interactive character cards with hover effects
- Upside Down gallery with eerie visuals
- Floating Eggo waffles animation
- Vecna's section with special effects
- Responsive design for all devices
- Dynamic lighting and hover effects

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Custom animations and transitions

## Setup

1. Clone the repository:
```bash
git clone https://github.com/Levon-Tumo/stranger-things-website.git
```

2. Open `index.html` in your web browser to view the website.

## Structure

- `index.html` - Main HTML structure
- `style.css` - All styling and animations
- `script.js` - Interactive features and effects
